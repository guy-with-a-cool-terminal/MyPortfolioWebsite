# Brian Njuguna - Portfolio

My personal portfolio website showcasing projects, skills, and experience as a software engineer.

## Built With

- React + TypeScript
- Vite
- shadcn/ui
- Tailwind CSS

## Features

- Project showcase with live demos and source code
- Technical skills and experience timeline
- Responsive design
- Modern UI components

## Local Development
```bash
# Clone the repository
git clone https://github.com/guy-with-a-cool-terminal/portfolio.git

# Install dependencies
npm install

# Start development server
npm run dev
```
